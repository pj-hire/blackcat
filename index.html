What is Blackcat

Blackcat is a simple CSS styling toolkit which can be utilized in styling a handful of HTML elements.

Layout

Blackcat's main layout consists of 'section-containers' and 'sections'. Having sections within section-containers comes in especially handy when in comes to adjusting the margin and alignment of elements. Below is the section you're reading now.

HTML:

  <div class="section-container">
    <div class="section">
      <div class="heading">
        <h2>Layout</h2>
        <p>Blackcat's main layout consists...</p>
      </div>
    </div>
  </div>

The Grid

The grid is a 12-column fluid grid, that shrinks with the browser/device at smaller sizes. The max width can be changed with one line of CSS and all columns will resize accordingly. The syntax is simple and it makes coding responsive much easier. Go ahead, resize the browser.

Typography

Type is all set with 'ems', so font-sizes and spacial relationships can be responsively sized based on a single HTML font-size property. All elements are base 16px, meaning that 1em is eqivelant to 16px, while 2em is equivilant to 32px and so on. Some basic elements included are:

  • anchors
  • strong
  • italic
  • underline
  • "Blockquote"

The typography base is Roboto served by Google, set at .9em (14.5px) over a 1.75em line height (28px).

The headings base is Montserrat, also served by Google. Headings create a family of distinct sizes each with specific letter-spacing, line-height, and margins.

Forms

Forms are all divided into rows, which can be split into samller sections using the grid. The most common way to divide rows is in half, using the "six_of_twelve" class. Rows with elements spanning the entire width of the container use the "twelve_of_twelve" class

Form Heading



Phone Text Email 

Forms are created with the 'form' tag and titled with the 'legend' tag. Start your form rows 
after the closing legend tag: 

  <form>
    <legend>
      <h5>
        Form Heading
      </h5>
    </legend>
    <div class="form-row">
    </div>
  </form>

Create two text inputs across the same row:

  <div class="form-row">
    <div class="six_of_twelve">
      <label for="text-input" class="text-label">Name:</label> <br>
      <input class="text-input" id="text-input" type="text" placeholder="Enter your name">
    </div>
    <div class="six_of_twelve">
      <label class="text-label" for="text-input">Email:</label> <br>
      <input class="text-input" id="text-input" type="text" placeholder="example@mailbox.com">
    </div>
  </div>

Create a dropdown menu across half a row:

  <div class="form-row">
    <div class="six_of_twelve">
      <label class="text-label" for="select-field">Select field:</label>
      <select class="text-input" id="select-field"> <br>
        <option selected="selected">Option 01</option>
        <option>Option 02</option>
        <option>Option 03</option>
      </select>
    </div>
    <div class="six_of_twelve-form">
    </div>
  </div>

Create a text-area input across the entire row:

  <div class="form-row">
    <div class="twelve_of_twelve">
      <label class="text-label" for="text-area">Message:</label> <br>
      <textarea class="text-area" id="text-area" name="text-area" cols="" rows="4" placeholder="Enter your message">
      </textarea>
    </div>
  </div>

Code

Displaying blocks of code can be cumbersome at times - Blackcat is here to make it as simple as possible. Wrap your code in <pre> tags with a "box" class, and make sure to use "<" in place of all your opening tags.

Code as it is displayed:

  <div class="section-container">
    <div class="section">
      <h2 class="main-heading">
        Hello World!
      </h2>
    </div>
  </div>

Code as it looks in your text editor:

  <div class="section-container">
    <div class="section">
      <h2 class="main-heading">
        Hello World!
      </h2>
    </div>
  </div>
